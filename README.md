# Student Performance Prediction
This project predicts student performance levels (High, Medium, Low) using machine learning models based on behavioral, academic, and engagement features.

## Dataset
- Source: Kaggle (xAPI-Edu-Data)
- Includes student demographics, participation, attendance, and parental involvement.

## Methods
- Data preprocessing (encoding + scaling)
- Train-test split (80/20)

### Models:
- Logistic Regression  
- Random Forest  
- Decision Tree  
- KNN  
- Neural Network  
---
## Results

| Model                | Accuracy |
|---------------------|---------|
| Logistic Regression | 79%     |
| Random Forest       | 80% ⭐   |
| Decision Tree       | 72%     |
| KNN                 | 76%     |
| Neural Network      | ~77–79% |
---
## Insights
- Higher engagement → better performance  
- More absences → lower performance  
- Parental involvement improves outcomes  
- Random Forest performed best
